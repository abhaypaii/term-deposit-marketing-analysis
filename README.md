# Bank Term Deposit Subscription Prediction

An end-to-end machine learning application that predicts whether a customer is likely to subscribe to a bank term deposit based on demographic information, financial attributes, and previous marketing campaign interactions. The project combines predictive modeling, explainable AI, and interactive visualizations to support data-driven marketing decisions and optimize customer targeting.

## Features

* **Subscription Prediction** – Predicts the likelihood of a customer subscribing to a term deposit using a trained machine learning model.
* **Probability Scoring** – Provides confidence scores for each prediction to support informed decision-making.
* **Customer Analytics Dashboard** – Interactive visualizations for exploring customer demographics, campaign performance, and subscription trends.
* **Feature Importance Analysis** – Identifies the most influential variables driving customer conversion.
* **Model Performance Evaluation** – Displays classification metrics including Accuracy, Precision, Recall, F1-score, ROC-AUC, and confusion matrix.
* **Real-Time Inference** – Streamlit-based interface for instant predictions using manually entered customer information.
* **Interactive Visualizations** – Plotly-powered charts for customer segmentation, feature distributions, and campaign insights.
* **Production-Ready Deployment** – Modular architecture with serialized models, reusable utility functions, and an interactive web application.
